Stage 19B - Branch 2

Emergent Transport-Corridor Reinforcement Probe

(Mesoscopic substrate law: corridor-selective persistence bias)

Role in the program

Stage 19A showed that local kernel back-reaction is bounded and regime-inert at weak strengths.
Stage 19B Branch 1 showed that basin-residence reinforcement produces real mesoscopic deformation but still no persistence gain or basin capture.

Branch 2 therefore changes the substrate law qualitatively:

-> Instead of reinforcing "being in a basin,"
this branch reinforces repeated transport pathways.

The diagnostic question is:

Can mesoscopic reinforcement of frequently used transport corridors produce
longer composite survival, stronger binding persistence,
or new stabilized ordering classes?

No geometric interpretation is allowed at this stage.

---

Substrate law: corridor reinforcement

The kernel is allowed to evolve through a bounded mesoscopic field
constructed from recent transport flow.

Define a corridor activity field C(x,t) such that:
- it accumulates where packet transport repeatedly occurs
- it decays on a finite memory horizon
- it remains bounded and smooth

Example conceptual form (not prescriptive):

∂_t C = beta * F_transport(phi) - (1/tau_c) * C

Kernel modification:

K -> K + delta * S(C)

where:
- delta is the reinforcement strength
- S is a smoothing / projection operator
- global normalization must be preserved
- no singular channel formation is permitted

This law differs from Branch 1:
- Branch 1 reinforced local residence density
- Branch 2 reinforces mesoscopic transport corridors

Pilot structure

Minimal early-stop design:
- 3 representatives
- 3 conditions each
- total base runs: 9

Conditions per representative:
- null (no corridor reinforcement)
- very weak reinforcement
- weak reinforcement

Base resolution:
- n = 12

Refinement:
- 12 -> 24 only if a run earns promotion.

---

Promotion criteria

A base run earns refinement only if all of the following occur:
1. measurable increase in composite lifetime vs null
2. increase in binding persistence or corridor survival
3. stable shift in ordering class or transport pattern
4. bounded kernel deformation (no runaway corridor collapse)

If any criterion fails -> no promotion.

---

Early negative freeze rule

Freeze Branch 2 negative if:
- all 9 base runs show
- zero composite lifetime gain
- zero binding persistence gain
- zero coarse basin persistence gain
- no new stabilized ordering class
- no corridor-selective transport locking

Even if:
- mesoscopic corridor field norm is nonzero
- kernel deformation is measurable

This still counts as a real negative pilot.

---

Expected diagnostic signatures

Possible weak positive
- longer survival of counter-propagating channels
- reduced fragmentation of transport structures
- selective stabilization in one representative only

Stronger candidate signal
- refinement-stable increase in corridor persistence
- reproducible shift of interaction regime classification
- bounded but structured kernel memory aligned with flow

What not to trust
- global stabilization across all regimes
- short-time survival spikes
- persistence gain without transport-pattern change
- refinement destroying the effect
- reinforcement norms growing while morphology remains unchanged

this architecture still lacks a substrate mechanism
capable of producing historical stabilization through transport feedback.
