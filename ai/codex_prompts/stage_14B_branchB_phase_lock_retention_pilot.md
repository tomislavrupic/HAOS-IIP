STAGE 14B — BRANCH B
Effective Binding Existence Test via Weak Phase-Lock Retention

Purpose
Test whether adding a weak memory-like retention of local phase alignment in overlap regions can improve collective composite persistence compared to the frozen Stage 13 baseline and the negative Stage 14A Branch A result.

Architecture rule
All frozen operator architecture elements remain unchanged.
The only new ingredient is a weak local phase-lock retention channel acting in overlap regions.

Baseline comparison
Every run must be paired with a null-control run using identical initial conditions and resolution.

Representatives
Use three previously identified morphologies:
1. tight clustered pair
2. asymmetric density cluster
3. counter-propagating corridor (diffuse control)

Conditions per representative
A. null control
B. very weak phase-lock retention
C. weak phase-lock retention

Total runs
3 representatives x 3 conditions = 9 runs

Metrics to compute
Collective layer
- composite lifetime
- centroid separation statistics
- overlap persistence time
- interaction label stability
- phase-lock indicator

Coarse layer
- dominant basin area fraction
- basin count stability
- coarse morphology classification

Scale response probe
For any run showing meaningful improvement over null, repeat at doubled resolution using the Stage 13 resolution ladder.

Decision gate
Stage 14B succeeds only if at least one representative shows:
- longer composite lifetime than null
- stronger phase-lock retention than null
- reduced softening under refinement
- basin stability increase beyond simple visual sharpening

Interpretation boundary
Results are read only as modification of superposition morphology and effective interaction persistence.
No force or physical binding claims.

Deliverables
- per-run trajectory and overlap plots
- coarse morphology regime map
- null vs retention comparison table
- short Stage 14B pilot note
