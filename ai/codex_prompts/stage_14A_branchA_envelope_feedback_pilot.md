STAGE 14A — BRANCH A
Effective Binding Existence Test via Weak Envelope Feedback

Purpose
Test whether adding a minimal envelope-feedback channel can produce measurable improvement in collective composite persistence compared to the frozen linear baseline established through Stage 13.

This stage does not search for new regimes.
It tests only whether effective binding signals begin to appear under the weakest structural extension.

Architecture rule
All frozen operator architecture elements remain unchanged.
The only new ingredient is a weak scalar envelope feedback term acting locally on packet amplitude evolution.

Baseline comparison
Every run must be paired with a null-control run using identical initial conditions and resolution.

Representatives
Use three previously identified morphologies:
	1.	tight clustered pair
	2.	asymmetric density cluster
	3.	counter-propagating corridor (diffuse control)

Conditions per representative

For each representative execute exactly three conditions:

A. null control
– frozen Stage 13 dynamics
– no feedback term

B. very weak feedback
– envelope feedback coefficient ε₁
– chosen such that single-packet dynamics remain visually unchanged

C. weak feedback
– envelope feedback coefficient ε₂ ≈ 2ε₁
– still within linear-like visual regime

Total runs
3 representatives × 3 conditions = 9 runs

Metrics to compute

Collective layer
– composite lifetime
– centroid separation statistics
– overlap persistence time
– interaction label stability

Coarse layer
– dominant basin area fraction
– basin count stability
– coarse morphology classification

Scale response probe
For any run showing lifetime increase over null, repeat at doubled resolution ladder using Stage 13 coupling rule.

Decision gate

Stage 14A succeeds only if at least one representative shows:

– statistically longer composite lifetime than null
– reduced softening under refinement
– basin stability increase beyond simple visual sharpening

If none satisfy this → terminate Branch A and freeze result as negative.

If one satisfies → promote only that representative to refinement follow-up.

Interpretation boundary

Results must be read strictly as:

– modification of superposition morphology
– emergence of effective interaction persistence

Not as evidence of nonlinear force laws or physical binding.

Deliverables

– per-run trajectory and overlap plots
– coarse morphology regime map
– null vs feedback comparison table
– short Stage 14A pilot note
