# Stage 11 - Integrated Austerity Resolution Probe

## Purpose

Attack the four open bridge obligations in one coupled experiment:

- `O_C0_ORIENT`
- `O_C0_EVOLUTION`
- `O_C0_GRADING`
- `O_C0_RECOVERY`

The probe forces mismatch evolution, signed combinatorial field update, harmonic-address sectoring, and topology survival to co-evolve in the same synchronous run so that any positive signal cannot be dismissed as dressing, surrogate bookkeeping, or post-hoc patching.

## Four Coupled Layers

1. sequencing layer  
   incidence-forced mismatch accumulation on a protected path

2. field-update layer  
   signed combinatorial update on the same graph complex

3. address-grading layer  
   static harmonic-address field acting as sector selector

4. topology-survival layer  
   braid / smear / capture class monitoring across matched graph families

## Mandatory Matched Families

- clustered braid seed
- corridor transport seed
- triad competition seed

All three must share:

- identical node count
- identical degree spectrum
- identical kernel width
- identical mismatch forcing amplitude

Only topology differs.

## Synchronous Step Cycle

At each step:

1. update the mismatch ledger
2. rebuild the signed combinatorial operator from the current mismatch orientation
3. apply the signed field update
4. apply intrinsic harmonic-address weighting
5. classify topology and survival on the same step

## Obligation Diagnostics

### ORIENT

`ledger_alignment = corr(protected_sequence_index, cumulative_mismatch_gradient)`

Positive if persistent and positive across at least two families.

### EVOLUTION

`delta_equiv = ||m_step_projection - signed_field_flux||`

Positive if bounded and refinement-stable.

### GRADING

`sector_entropy = entropy(topology_class | address_band)`

Positive if entropy falls under refinement and detuning.

### RECOVERY

`P_family = mean_survival_time(protected_paths)`

Positive if the ordering

`clustered >= corridor >= triad`

survives detuning, degree-skew, and motif panels.

## Scan Structure

Use a compact but complete grid:

- 3 families
- 3 detuning samples
- 3 protection strengths

Kernel radius `r-1, r, r+1`, refinement doubling, degree skew, and motif injection are evaluated synchronously inside each run.

## Expected Global Outcomes

- `strong_resolution`
- `partial_structuralization`
- `negative_resolution`

## Atlas Sentence

Stage 11 tests whether incidence-forced sequencing, signed combinatorial evolution, intrinsic harmonic-address sectoring, and topology survival form a single coherent dynamical structure. A positive result would indicate that harmonic address functions as an intrinsic grading rather than a selector dressing; a negative result would confine address effects to topology-selection phenomenology.
