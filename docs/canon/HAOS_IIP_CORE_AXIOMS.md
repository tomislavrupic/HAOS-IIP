# HAOS_IIP_CORE_AXIOMS

Status: canonical core statement

This file defines the frozen HAOS-IIP core.

It is prior to any specific kernel choice, operator lift, numerical branch, or experimental realization.

Model families, simulations, and appendices may test these axioms inside particular representational regimes, but they do not revise the core unless this file is changed explicitly.

## Core axioms

### Axiom 1 - Recoverability

A configuration is real if and only if it can recover non-zero coherence after any bounded perturbation.

Configurations that cannot are excluded from operational reality.

### Axiom 2 - Interaction Primacy

All structure arises from interaction.

No entity, property, or relation exists independently of interaction.

### Axiom 3 - Representational Regimes

Descriptions of reality exist only within representational regimes `Pi`, each a compression of admissible interaction histories.

No regime is ontologically privileged.

### Axiom 4 - Viability-Constrained Dynamics

Any admissible evolution must preserve recoverability within its `Pi`.

When recoverability fails in a regime, the representation contracts to the next viable lower-order `Pi`.

### Axiom 5 - Invariant Residue

Upon contraction, only interaction invariants persist.

Particles, spacetime, and classical objects are invariant residues stabilized within specific `Pi`-regimes.

## One-line closure

Reality is what survives interaction under recoverability constraints; all descriptions are secondary compressions of that survival.

## Boundary note

This core statement is for AI, tooling, and theory organization as much as for human readers.

Interpretation rule:

- core axioms define HAOS-IIP
- model families realize the axioms inside specific regimes
- experiments test those realizations
- negative or positive experimental results do not, by themselves, rewrite the core
